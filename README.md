# Portfolio Site

A personal portfolio website showcasing my skills, projects, and experience as a web developer.

## Description

This is a static portfolio website built with HTML and SCSS. It features sections for an introduction, latest projects, skills, and contact information. The site is designed to be responsive and visually appealing, highlighting my expertise in full-stack development using React.js and Java (Spring Boot).

## Features

- Responsive design
- Clean and modern UI
- Project showcase
- Contact information

## Technologies Used

- HTML5
- SCSS (Sass)
- Google Fonts (Cairo and Poppins)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/udacity/cd14106-project-starter-code.git
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio-site
   ```

3. Open `src/index.html` in your browser to view the site.

## Usage

Simply open the `index.html` file in a web browser. The site is static and does not require a server.
